# ⚡ AI-Based Transformer Health Prediction System

## 📌 Overview
This project is a machine learning-based predictive maintenance system that analyzes transformer operational data and predicts equipment health status.

## 🚀 Features
- Predicts transformer health (Healthy / Warning / Critical)
- Uses Random Forest Machine Learning algorithm
- Flask-based web application
- Real-time user input and prediction

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas, NumPy

## 📊 Input Parameters
- Oil Temperature
- Winding Temperature
- Load Current
- Voltage
- Ambient Temperature
- Moisture
- Dissolved Gases (H2, CO, CH4)
